Большая подборка плагинов на чистом JS для разных задач

Привет! Ну, пока данная подборка не&nbsp;будет действительно большой, я&nbsp;лишь начинаю ее&nbsp;заполнять. Однако, постепенно все будет) Вы&nbsp;можете предлагать свои варианты в&nbsp;комментариях, и&nbsp;возможно я&nbsp;их&nbsp;добавлю.

1. Слайдер

swiper.js — прекрасный плагин для слайдера.

https://youtu.be/OjVK055CTNI — мой туториал по данному плагину

2. Валидация форм

just-validate — простой плагин для валидации. Очень схож с jquery-validation.

https://youtu.be/EPXabErmGrQ — мой туториал по данному плагину

3. Маскирование

inputmask.js — плагин для маскирования. Отлично подойдет для маски телефона.

https://youtu.be/h5fSz30CLQ4 — мой туториал по данному плагину

4. Focus-trap плагины

inert.js — не совсем плагин, но удобный полифилл для «поимки» скролла.

https://youtu.be/jmHRLWeugKc — мой туториал по данному плагину

5. Кастомный скролл

simplebar.js — простейший и по коду, и по использоваю плагин для кастомного скролла. https://youtu.be/C_Q6g1lGpDU — мой туториал по данному плагину

6. Кастомный селект

choices.js — плагин для создания кастомного селекта с кучей функций. https://youtu.be/dnC7XCYb9Qg — мой туториал по данному плагину

7. Range слайдер

nouislider.js — плагин для создания разнообразных range-слайдеров. https://youtu.be/kWoAKq7N2Ew — мой туториал по данному плагину

8. Lightbox

fslightbox.js — простой аналог fancybox.js для создания lightbox-галерей. https://youtu.be/QajyOqPeZw8 — мой туториал по данному плагину

9. Модальное окно

graph-modal.js — мой плагин для модальных окон, крайне простой и легкий. https://youtu.be/jeKw5n3F8nw — мой туториал по данному плагину

10. LazyLoad

lazyload.js — плагин для отложенной загрузки изображений.

11. Работа с cookie

js-cookie.js — простой плагин для управления cookie на сайте. https://youtu.be/RrMgCWBtE_s — мой туториал по данному плагину

12. Работа со свайпами

swiped-events.js — плагин для отслеживания свайпов.

https://youtu.be/VoxDjhCqqqE — мой туториал по данному плагину

13. Анимация по скроллу

AOS.js — плагин анимации по скроллу.

https://youtu.be/vrEHi8kp8M4 — мой туториал по данному плагину

14. Увеличение изображения

js-image-zoom.js — плагин для увеличения изображения.

https://youtu.be/VIge0P410Jw — мой туториал по данному плагину

15. Обрезание многострочного текста

ellipsis.js — плагин для обрезания многострочного текста.

https://youtu.be/MUZvekTFH-4 — мой туториал по данному плагину
